# ir-schema

JSON Schema & TypeScript types untuk IR Visual Design & Multimedia DSL.

- Mendefinisikan schema IRDocument, IRNode, IRCanvas, dsb.
- Menyediakan validator berbasis AJV
- Digunakan oleh ir-compiler dan renderer

## Struktur
- `src/` — source code schema, types, validator
- `test/` — (akan dibuat) unit test untuk validasi schema

## Instalasi
```bash
npm install
```

## Penggunaan
Akan tersedia setelah implementasi schema dan validator.
