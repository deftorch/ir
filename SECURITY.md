# Security Policy

## Supported Versions

We actively support and patch security issues in the following versions:

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not open a public issue. Instead, report it through the following steps:

1. Send an email to the project maintainers detailing the vulnerability.
2. Include reproduction steps, potential impact, and system environment info.
3. We will acknowledge your report within 48 hours and provide a timeline for a resolution.
4. Once resolved, we will publish a security advisory and issue an update.
